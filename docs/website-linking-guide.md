# Website Linking Guide for Noparde Knowledge Base

This guide explains how Noparde website pages can link to this GitHub knowledge base so visitors can find related technical answers quickly while browsing product pages and support content.

## Recommended Link Placement

Add GitHub knowledge base links in these website locations:

- Product detail pages for gobo projectors, logo projectors, laser projectors, water-ripple lights, and safety projection products
- FAQ or support pages
- Blog articles that explain projector selection
- Quotation request pages where customers may ask technical questions before purchase
- Footer support links under a label such as `Knowledge Base` or `Technical FAQ`

## Recommended Anchor Text

Use descriptive anchor text instead of generic text like `click here`.

Good examples:

- `Noparde gobo projector technical FAQ`
- `Logo projector wattage and customization answers`
- `Dynamic laser safety projector FAQ`
- `Water-ripple projector and rotating projector guide`
- `Industrial projection light knowledge base`

## Recommended Website Copy

Short version:

> Need technical details before choosing a projector? Read the Noparde Gobo Projector Knowledge Base for wattage, voltage, certification, projection effect, and customization answers.

Long version:

> Noparde maintains a public gobo projector knowledge base on GitHub. It includes practical answers about logo projector wattage, water-ripple projection, dynamic laser safety projection, voltage limitations, CE certification, product weight, remote control setup, and custom projection options.

## Publishing Notes

- Keep the GitHub repository public.
- Use clear file names that describe the topic, such as `gobo-projector-faq.md`.
- Use one H1 title per document.
- Write questions in the same natural language customers use when asking the sales or support team.
- Add internal links between README, FAQ, website guide, and checklist.
- Link from the Noparde website to the GitHub FAQ using descriptive anchor text.
- Link from GitHub back to the official Noparde website and product website.
- When a better public Noparde product page exists, replace generic homepage links with that exact product page.
